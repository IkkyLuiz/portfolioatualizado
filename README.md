## 🚀 Como executar meus projetos localmente

1. Clone o repositório:

   ```bash
   git clone https://github.com/IkkyLuiz/<nome-do-projeto>.git
````

2. Acesse a pasta do projeto:

   ```bash
   cd <nome-do-projeto>
   ```

3. Instale as dependências:

   ```bash
   # Exemplo para Node.js
   npm install
   ```

4. Configure variáveis de ambiente (se necessário)

5. Execute o projeto:

   ```bash
   # Exemplo para Node.js
   npm start
   ```

> Observação: veja o `README` de cada projeto para instruções específicas.
